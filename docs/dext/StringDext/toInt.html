toInt method Null safety

int toInt(
  1. {int radix = 10}
)

Converts string to an integer.

If radix is not specified, the radix is 10.

Such as:

  • '123'.toInt() => 123
  • 'FF'.toInt(radix: 16) => 255
  • '1011'.toInt(radix: 2) => 11

Implementation

int toInt({
  int radix = 10,
}) {
  return int.parse(this, radix: radix);
}
